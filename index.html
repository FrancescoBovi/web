Web App Specification Language

A programming language for building web apps.

Get StartedTake the Tutorial

Wasp is a declarative language for buildingÂ full-stack web apps with less code.

Front-end, back-end and deployment - all in one concise DSL.

How it works

Given .wasp files as an input, Wasp generates the full source code of your web app - front-end, back-end and deployment.

Wasp is used along with files written in standard web technologies - .js(x),.css, ...

Quick start

Create a production-ready web app from scratch with only a few lines of concise, declarative code.

Flexible

Move fast using high-level constructs provided by Wasp or drop down to js/html/css... when you require more control.

No lock-in

If Wasp becomes too limiting for you, simply eject and continue with the source code following industry best-practices, as if it was handwritten by a senior engineer.

The Language

Concepts such as app, page, route, authÂ  etc. are baked into Wasp, providing the higher level of expressiveness.

todoApp.wasp
app todoApp {
title: "ToDo App"
}
route "/" -> page Main
page Main {
component: import Main from "@ext/Main"
}
ext/Main.js
import React from 'react'
export default () => <span> Hello World! </span>
That's it, this is the whole app! Run wasp start and check it out at http://localhost:3000!
