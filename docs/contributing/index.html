Contributing

Any way you want to contribute is a good way, and we are happy to meet you :)!

Some typical ways to contribute:

  1. Join us on discord and let's talk! We can discuss language design, new/existing features, weather, or you can just tell us how you feel about Wasp :).
  2. If there is something in docs that you think could be improved or clarified better, go to docs Github repo and make an issue/PR!
  3. Create an issue/PR on core Wasp repo to contribute directly to the language and/or compiler! Check wapsc README for more details.
